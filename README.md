# PRODIGY_DS_05

## Task 05: Traffic Accident Data Analysis

### Objective
Analyze traffic accident data to identify patterns related to road conditions, weather, and time of day.

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Kaggle Notebook

### Visualizations
- Accidents by Hour
- Weather Condition Analysis
- Severity Distribution
- Temperature Distribution
- Correlation Heatmap
- Accident Hotspots

### Outcome
Identified accident trends based on weather, accident severity, and time of day using data visualization.
